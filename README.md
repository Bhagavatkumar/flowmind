## 🚀 How to Deploy (Free — 3 Minutes)

### Option A: Netlify Drop (Recommended — fastest)
1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag your entire `flowmind/` folder onto the page
3. Get a live URL instantly (e.g., `flowmind-abc123.netlify.app`)
4. (Optional) Set a custom subdomain like `flowmind-app.netlify.app`

### Option B: GitHub Pages
1. Create a new GitHub repo named `flowmind`
2. Upload all files maintaining the folder structure
3. Go to Settings → Pages → Source: Deploy from branch (main)
4. Your URL: `https://yourusername.github.io/flowmind`

### Option C: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Inside the project folder: `vercel --prod`
3. Get instant live URL

## 📝 MeDo Submission Description Template

> **Problem Solved:**
> Most people start their day with a mental fog — too many tasks, no clear priority, and no system to stay accountable. FlowMind solves this by combining a visual daily planner with an AI coach that gives real-time advice, helping users focus on what matters most.
>
> **How I talked to MeDo:**
> I started by describing the overall product: "Build a dark-themed productivity app with a sidebar navigation, a today view showing time-blocked tasks, a kanban board, and an AI chat section." MeDo generated the initial scaffold.
>
> Then I iterated: "Make the focus score a circular ring chart that animates on load." Then: "Add drag-and-drop between kanban columns." Then: "Create an analytics view with a sparkline, a productivity heatmap, and category bars."
>
> The AI coach section came from: "Add an AI chat panel where the bot responds to prompts like 'plan my morning' or 'I'm feeling overwhelmed' with specific, helpful advice."
>
> The most powerful moment was when I said: "Make the whole thing feel like a premium SaaS product — use a dark color scheme with an amber accent, the Syne font for headings, and smooth animations throughout." MeDo understood the aesthetic vision and executed it.
